``casting`` and ``dtype`` keyword arguments for `numpy.stack`
-------------------------------------------------------------
The ``casting`` and ``dtype`` keyword arguments is now available for `numpy.stack`. 
To use it, write ``np.stack(..., dtype=None, casting='same_kind')``.
